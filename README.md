# Садовник (The Gardener)

Многопользовательское десктопное приложение-игра, разработанное с нуля на C# и OpenGL (без использования готовых игровых движков типа Unity/Unreal).

Проект демонстрирует навыки проектирования архитектуры ПО, работу с низкоуровневой графикой и применение паттернов GOF.

## О проекте
Игроки управляют садовниками, соревнуясь в выращивании деревьев (яблони и груши). Необходимо вовремя поливать растения, удобрять их, лечить от вирусов/грибков и защищать от вредителей, используя общий инвентарь. Побеждает тот, кто соберет больше урожая.

## Технологии и Архитектура
*   **Язык:** C# (.NET Framework)
*   **Графика:** OpenGL 3.3 (через библиотеку OpenTK) — ручная отрисовка спрайтов, настройка матриц проекции, GLSL шейдеры (опционально).
*   **UI:** Windows Forms (интеграция GLControl).
*   **Аудио:** NAudio / WMPLib.

### Примененные паттерны проектирования (ООП)
В проекте реализована строгая объектно-ориентированная архитектура:
1.  **Фабричный метод (Factory Method):**
    *   Используется для генерации случайных проблем у деревьев (`AppleTreeProblemFactory`, `PearTreeProblemFactory`). Позволяет легко добавлять новые типы угроз (засуха, жуки) без изменения основного кода.
2.  **Декоратор (Decorator):**
    *   Классы `AppleTreeDecorator` и `PearTreeDecorator` расширяют базовый класс `Tree`. Это позволяет динамически наделять деревья уникальными свойствами и вероятностями возникновения болезней.
3.  **Игровой цикл (Game Loop):**
    *   Реализован кастомный цикл обновлений (`GameManager.Update`) и отрисовки (`GameRenderer.Render`), разделяющий логику и представление.

## Геймплей и Механики
*   **Мультиплеер (Hot-seat):** Два игрока за одной клавиатурой (WASD vs Стрелки).
*   **Управление ресурсами:** Инструменты в инвентаре (лейка, спреи) — общий ресурс. Механика блокировки (mutex-like) не позволяет игрокам брать один предмет одновременно.
*   **Симуляция:** Деревья имеют стадии роста, здоровье и потребности, возникающие по таймерам.

## Как запустить
1.  Клонируйте репозиторий.
2.  Откройте решение `Gardener.sln` в **Visual Studio 2019/2022**.
3.  Убедитесь, что установлены пакеты NuGet: `OpenTK`, `NAudio`.
4.  Соберите проект (Build -> Build Solution).
5.  Запустите `Gardener.exe`.
